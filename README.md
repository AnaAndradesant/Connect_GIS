🔌 Connect: Navegação Inteligente de Camadas para QGIS
O Connect é uma ferramenta de produtividade desenvolvida para usuários do QGIS que lidam com projetos densos e complexas árvores de camadas. Ele elimina a necessidade de marcar e desmarcar grupos manualmente, permitindo uma alternância fluida e focada.

📌 O que ele faz
O plugin isola a visualização de grupos específicos. Ao selecionar um grupo através do painel Connect, ele ativa automaticamente as camadas daquele grupo e oculta o grupo anterior, garantindo que você visualize apenas um contexto por vez.

🚀 Principais Funcionalidades
Gestão por Contexto: Adicione qualquer grupo da legenda ao painel do plugin com um simples clique no botão direito.

Navegação por Teclado: Utilize as setas ↑ (Cima) e ↓ (Baixo) para alternar entre os grupos selecionados, ideal para comparações rápidas de mapas.

Controle Centralizado: Botões de "Anterior" e "Próxima" para um fluxo de trabalho sequencial.

Atalho Rápido: Pressione F9 para exibir ou ocultar o painel do Connect instantaneamente.

Interface Intuitiva: Suporte a Drag-and-Drop para reordenar a sequência de exibição dos grupos conforme sua necessidade.

Memória do Projeto: O plugin salva a lista de grupos configurada dentro do arquivo .qgs/.qgz, para que você não precise reconfigurar tudo ao abrir o projeto novamente.

🎮 Guia Rápido de Uso
Na árvore de camadas (Legenda), localize o grupo desejado.

Clique com o botão direito sobre o grupo e escolha: Connect: Adicionar grupo.

Abra o painel do Connect (se estiver fechado, use F9).

Navegue entre os itens da lista. O QGIS gerenciará a visibilidade dos grupos automaticamente para você.

⚙️ Requisitos Técnicos
Versão do QGIS: 3.22 (Bialowieza) ou superior.

Linguagem: Desenvolvido em Python com interface PyQt5.
